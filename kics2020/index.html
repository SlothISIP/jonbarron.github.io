Abstract

This paper proposes a novel approach using Z-domain transformation to extract frequency-based features for robust respiratory status monitoring. By analyzing UWB radar signals in the Z-domain, we achieve reliable classification of normal breathing, abnormal patterns, and apnea events.

Key Contributions

  • Z-Domain Analysis: Novel application of Z-transform for respiratory signal characterization
  • Pole-Zero Analysis: Extraction of discriminative features from pole-zero patterns
  • Frequency Features: Robust frequency-domain descriptors for respiration classification
  • Real-time Processing: Efficient implementation suitable for continuous monitoring

Methodology

The Z-domain analysis approach:

  1. UWB radar signal acquisition and preprocessing
  2. Windowed Z-transform computation on respiratory segments
  3. Pole-zero location extraction and analysis
  4. Feature vector construction from Z-domain characteristics
  5. Classification using machine learning algorithms
