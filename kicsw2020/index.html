Abstract

This paper presents classification of respiratory status using UWB radar signals with machine learning algorithms for non-contact health monitoring. We explore various learning approaches to automatically distinguish between normal breathing, shallow breathing, and apnea events from radar-derived respiratory waveforms.

Key Contributions

  • Feature Engineering: Comprehensive feature extraction from UWB radar respiratory signals
  • ML Classification: Comparison of multiple learning algorithms for respiratory status
  • Non-contact Monitoring: Radar-based approach without physical sensors on the subject
  • Multi-class Classification: Distinguishing multiple respiratory states including apnea

Methodology

The machine learning pipeline:

  1. UWB radar signal acquisition during different respiratory states
  2. Signal preprocessing and respiratory waveform extraction
  3. Feature extraction: time-domain, frequency-domain, and statistical features
  4. Training and evaluation of multiple classifiers (SVM, Random Forest, etc.)
  5. Model selection based on accuracy and real-time performance
