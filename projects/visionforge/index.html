VisionForge: AI Vision Platform
for Industrial Inspection Production-Grade PCB Defect Detection

  • Ju O Kim
    Keimyung University

Overview

VisionForge is a production-grade AI vision platform designed for PCB (Printed Circuit Board) defect inspection in manufacturing environments. The system supports IPC-A-610 compliant defect categories through a multi-model architecture combining YOLOv8, Stable Diffusion, and NeRF.

Note: Code available upon request for academic collaboration.

System Architecture


End-to-end PCB inspection pipeline architecture showing the flow from high-resolution camera input through YOLOv8 detection engine to IPC-A-610 compliance classification. The system incorporates Stable Diffusion for synthetic defect augmentation, NeRF+Open3D for 3D reconstruction, and FastAPI for seamless integration.

Defect Detection Dashboard


The main dashboard provides a comprehensive view of PCB inspection results, displaying detected defects with bounding boxes, confidence scores, and defect classifications in real-time. The interface supports batch processing and individual board analysis with IPC-A-610 compliance checking.

Macro Closeup Analysis


High-resolution macro closeup view enables detailed inspection of individual components and solder joints, supporting precise defect identification at the microscopic level. The system analyzes solder bridges, insufficient solder, cold joints, and component placement errors.

Live Inspection Interface


The live inspection interface provides real-time feedback during production, with instant defect detection overlays and pass/fail status indicators for seamless quality control integration. GPU acceleration enables processing speeds suitable for high-throughput production lines.

Multi-Board Batch Processing


Batch processing mode enables efficient inspection of multiple PCBs simultaneously, with aggregated statistics and defect distribution analysis across production batches. Quality metrics and trend analysis help identify systematic production issues.

Model Training Evolution


Visualization of the model training process showing loss curves, accuracy metrics, and performance evolution over training epochs. The system uses Stable Diffusion for synthetic data augmentation and NeRF for 3D reconstruction to enhance training data quality.

Technical Specifications

3D Reconstruction NeRF, Open3D
