Overview

VisionForge is a production-grade AI vision platform designed for PCB (Printed Circuit Board) defect inspection in manufacturing environments. The system supports over 60 defect types with IPC-A-610 compliance and provides real-time quality assurance capabilities.

Inspection Pipeline


The inspection pipeline follows a 4-stage process: Image Acquisition with calibration support, Preprocessing for enhancement and normalization, Defect Detection using YOLOv8 with multi-stage analysis, and Classification following IPC-A-610 standards for pass/fail determination. The system targets 95-98% detection accuracy.

Supported Defect Types (60+)

Solder Defects

  • Solder Bridge
  • Insufficient Solder
  • Excess Solder
  • Cold Solder
  • Solder Balls
  • Solder Voids

Component Defects

  • Missing Component
  • Tombstoning
  • Wrong Polarity
  • Misalignment
  • Wrong Component
  • Lifted Lead

PCB Defects

  • Scratches
  • Contamination
  • Delamination
  • Trace Damage
  • Via Defects
  • Pad Lifting

Key Features

  • 60+ Defect Types: Comprehensive coverage including solder bridges, missing components, tombstoning, and more
  • IPC-A-610 Compliance: Follows industry standard acceptance criteria for electronic assemblies
  • Multi-Model Architecture: Integrates YOLOv8 for detection, Stable Diffusion for synthetic data, and NeRF for 3D reconstruction
  • Production Ready: FastAPI backend with real-time inference capabilities
  • Modular Design: Plugin architecture for custom defect types and inspection workflows

Performance

The system targets 95-98% detection accuracy on standard PCB defect benchmarks. Performance varies based on defect type, image quality, and hardware configuration. GPU acceleration (NVIDIA RTX series recommended) enables real-time inference for production line integration.
