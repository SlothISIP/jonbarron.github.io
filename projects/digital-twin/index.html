Overview

The AI Digital Twin Factory System is an integrated software architecture for smart manufacturing combining vision AI, SimPy-based digital twin simulation, predictive maintenance, and production scheduling. The microservices architecture enables real-time monitoring and optimization of manufacturing processes through a React dashboard.

Note: Code available upon request for academic collaboration.

System Architecture


Microservices architecture with centralized API Gateway connecting five core modules: Vision AI for defect detection, Digital Twin for SimPy-based simulation, Predictive Maintenance for anomaly detection, Production Scheduling for resource optimization, and Real-time Monitoring via React Dashboard. Infrastructure runs on Docker/Kubernetes with PostgreSQL and RabbitMQ.

Microservices Architecture Diagram


Comprehensive microservices architecture showing the interconnection between Vision AI, Digital Twin, Predictive Maintenance, Production Scheduling, and Monitoring modules through the central API Gateway. Each service is independently deployable and scalable via Docker/Kubernetes.

Factory Monitoring Dashboard


Real-time factory monitoring dashboard displaying live KPIs, machine status indicators (green=running, yellow=idle, red=down), production metrics, and alert notifications. The React-based interface provides comprehensive operational oversight with OEE tracking.

API Documentation Interface


Interactive API documentation interface built with FastAPI/Swagger, providing clear endpoint definitions, request/response schemas, and testing capabilities for all microservices. Enables seamless integration with external systems and third-party applications.

SimPy Event Processing Visualization


Discrete-event simulation visualization showing the SimPy-based event queue processing, machine state transitions, and production flow modeling within the digital twin environment. Enables what-if analysis and process bottleneck identification.

Machine Performance Analysis


Detailed machine performance analysis dashboard showing OEE breakdown, availability trends, performance metrics, and quality indicators for individual equipment and production lines. Supports predictive maintenance scheduling and resource optimization.
