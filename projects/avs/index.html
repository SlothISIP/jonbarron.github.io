Overview

AVS is an end-to-end multimodal AI platform combining acoustic imaging, computer vision, and deep learning for industrial machine anomaly detection. The project provides complete code infrastructure and trained baseline models for acoustic vision research.

System Architecture


The AVS architecture integrates multiple signal processing and deep learning components. Audio signals from microphone arrays are processed through beamforming algorithms (Delay-and-Sum, MVDR, MUSIC) for sound source localization. Visual data from cameras enables object detection and depth estimation. The multimodal fusion layer combines acoustic and visual features using Uncertainty-Aware Multimodal Fusion (UAMF) for robust anomaly detection.

Key Features

  • Signal Processing: Delay-and-Sum, MVDR, MUSIC beamforming; GCC-PHAT localization; Spectral Subtraction, Wiener Filtering
  • Computer Vision: YOLOv8 object detection, MiDaS depth estimation
  • Anomaly Detection: AST (Audio Spectrogram Transformer) + LoRA + K-NN scoring
  • Multimodal Fusion: UAMF (Uncertainty-Aware Multimodal Fusion)

Technology Stack

  • Deep Learning: PyTorch 2.0+, Transformers, PEFT (LoRA)
  • Audio: torchaudio, librosa
  • Vision: Ultralytics YOLOv8, timm (MiDaS)
  • Deployment: ONNX export, GPU optimized inference
