Overview

ITS_AI_XAI is a production-ready AI system for PCB manufacturing quality control. The system combines sensor data analysis with computer vision for early defect detection, root cause analysis, and process optimization using explainable AI techniques.

System Architecture


The multimodal fusion pipeline combines sensor data (603 features) with visual PCB inspection. A Random Forest model processes sensor data while YOLOv11 handles visual defect detection. The Meta-Classifier (XGBoost) fuses both modalities, achieving F1-score of 0.9462. SHAP and Counterfactual explanations provide interpretability for manufacturing engineers.

Key Capabilities

  • Defect Early Warning: Real-time defect prediction during production (before final inspection) with 95.91% recall - catches 96% of defects
  • Root Cause Analysis: SHAP-based feature importance identification and counterfactual explanations ("what-if" scenarios)
  • Quality Process Optimization: Statistical pattern analysis with actionable recommendations for parameter tuning

SHAP Waterfall Analysis


SHAP (SHapley Additive exPlanations) waterfall plot showing how each feature contributes to the final prediction. Red arrows indicate features pushing toward DEFECT, while blue arrows show features pushing toward PASS. For this sample, Temperature_Zone3 (+0.18) and Solder_Paste_Volume (+0.12) are the primary drivers of the defect prediction.

Counterfactual Explanation


The counterfactual explanation shows the minimum changes needed to flip the prediction from DEFECT to PASS. For this example, reducing temperature from 248°C to 235°C and increasing pressure from 1.8 to 2.1 bar would change the prediction to PASS with 71.2% confidence. This provides actionable insights for process engineers to adjust manufacturing parameters.
