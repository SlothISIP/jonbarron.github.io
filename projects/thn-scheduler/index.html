Overview

THN AI Production Scheduler is an intelligent production planning system for wire harness manufacturing. The system optimizes scheduling for 67,738 circuits across 21 machine groups using 15 different algorithms, achieving 22% improvement in load balance (CV 0.35) and 32% reduction in applicator changes (APP 120).

Note: Code available upon request for academic collaboration.
