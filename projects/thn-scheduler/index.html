THN AI Production Scheduler Intelligent Production Planning System

  • Ju O Kim
    Keimyung University

Overview

THN AI Production Scheduler is an AI-powered production scheduling system designed for manufacturing optimization. The system implements multi-objective optimization algorithms for resource allocation and workflow management in industrial settings, with an interactive dashboard for real-time monitoring and control.

Dashboard Interface


The main scheduler dashboard provides a comprehensive view of production planning. Users can configure scheduling parameters, view current job status, and monitor resource utilization in real-time. The interface supports multiple scheduling modes and constraint configurations.

Production Gantt Chart Visualization


Enterprise-grade Gantt chart visualization shows the complete 5-day production schedule (December 8-12) across all 21 machine groups (M01-M21) managing 67,738 wire harness circuits. Task bars are color-coded by wire gauge (SQ range 0.22-3.0mm²) with a gradient from green (small gauge) to red (large gauge circuits). Vertical red dashed markers (▼APP) indicate applicator changes requiring setup time. The left panel displays machine load percentages with capacity bars, while the dark theme with neon cyan accents provides optimal visibility for factory floor displays. Key metrics in the header show CV (Load Balance) of 0.35 representing 22% improvement over production baseline, APP (Applicator Changes) of 120 reflecting 32% reduction in setup time, and total Makespan of 14,250 minutes for completing all circuits.

5-Day Optimization Results


The 5-day scheduling optimization comparison shows comprehensive performance metrics across six dimensions: Load Balancing Performance (CV reduction), Tool Change Efficiency (APP optimization), Circuit Reassignments (flexibility), Daily Workload Distribution (makespan per day), Overall Equipment Effectiveness (OEE), and Quality Metrics. The dashboard compares Step05 baseline, optimized algorithm results, and Step06 final schedule, demonstrating significant improvements in load balance (CV from 0.46 to 0.36), setup time reduction (APP from 150 to 120 changes), and overall completion time optimization across all 21 machines.

Algorithm Comparison


Performance comparison across 15 scheduling algorithms evaluating load balance (CV), setup changes (APP), and completion time (Makespan). The comparison highlights top-performing algorithms including ALNS (Adaptive Large Neighborhood Search) with CV 0.35 and best overall quality, Two-Phase Balanced achieving perfect load balance (CV 0.00), and CP-SAT providing optimal solutions for smaller datasets. The visualization enables data-driven algorithm selection based on specific production priorities such as minimizing setup time, balancing workload, or reducing total completion time.

Key Features

  • Multi-Objective Optimization: Balances multiple production goals (throughput, cost, quality, due dates)
  • Resource Allocation: Optimizes assignment of machines, workers, and materials
  • Workflow Management: Sequences production tasks for maximum efficiency
  • Constraint Handling: Respects machine capabilities, labor rules, and delivery deadlines
  • Real-time Dashboard: Interactive visualization of schedules and metrics

Scheduling Capabilities

Scheduling Type Description
Job Shop Optimal sequencing for multiple jobs across different machines
Flow Shop Sequential processing optimization for linear workflows
Flexible Manufacturing Dynamic routing and machine selection
Batch Processing Grouping and scheduling of batch operations

Optimization Objectives

  • Minimize makespan (total completion time)
  • Minimize tardiness (late deliveries)
  • Maximize machine utilization
  • Minimize setup times and changeovers
  • Balance workload across resources

Technology Stack

  • Backend: Python, FastAPI
  • Frontend: Streamlit Dashboard
  • Optimization: Custom scheduling algorithms, heuristics
  • Visualization: Plotly, Gantt charts, resource utilization plots
