Abstract

This paper presents an optimized boundary and corner detection method tailored specifically for thermal images to improve calibration accuracy in multi-camera systems. Thermal images pose unique challenges due to lower contrast and different noise characteristics compared to visible-light imagery.

Key Contributions

  • Thermal-Specific Detection: Algorithm optimized for thermal image characteristics
  • Enhanced Edge Detection: Adaptive thresholding for low-contrast thermal boundaries
  • Sub-pixel Corner Refinement: Accurate corner localization in thermal imagery
  • Multi-Camera Support: Consistent detection across multiple thermal camera views

Methodology

The detection pipeline:

  1. Thermal image preprocessing with contrast enhancement
  2. Adaptive Canny edge detection with thermal-specific parameters
  3. Line fitting using Hough transform
  4. Corner detection at line intersections
  5. Sub-pixel refinement using gradient-based optimization
