Abstract

This paper proposes isolating respiratory patterns from complex signal mixtures using high-pass filtering for more accurate apnea detection. When multiple subjects or noise sources are present, the respiratory signal becomes contaminated, making abnormal event detection challenging.

Key Contributions

  • Signal Separation: High-pass filtering to isolate respiratory components from mixed signals
  • Noise Robustness: Improved detection accuracy in noisy environments
  • Apnea Detection: Enhanced sensitivity to abnormal breathing patterns
  • Real-time Processing: Efficient filtering suitable for continuous monitoring

Methodology

The signal processing pipeline:

  1. Raw signal acquisition from UWB radar
  2. High-pass filter design with cutoff optimized for respiratory frequencies
  3. Signal filtering to remove low-frequency drift and DC components
  4. Feature extraction from filtered respiratory waveform
  5. Classification of normal vs. abnormal respiration status
