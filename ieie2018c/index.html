Abstract

This paper presents a camera calibration method using learning-based corner and edge detection techniques. By replacing traditional feature detection with deep learning models, we achieve more robust and accurate detection of calibration targets under challenging imaging conditions.

Key Contributions

  • Learning-Based Detection: Deep learning model for calibration pattern feature detection
  • Robustness: Improved detection under varying lighting and noise conditions
  • Accuracy: Sub-pixel accurate corner localization using learned features
  • Generalization: Applicable to various calibration pattern types

Methodology

The learning-based calibration approach:

  1. Training data generation from synthetic and real calibration images
  2. CNN model training for corner and edge detection
  3. Inference on calibration board images
  4. Post-processing for sub-pixel corner refinement
  5. Standard camera calibration with detected features
